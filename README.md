# Data Cleaning and Preparation

This repository contains the solution for **Task 02** of the SkillCraft Technology Data Analytics Internship.

The objective of this project is to clean and prepare a raw dataset using **Python** and the **Pandas** library.

---

## Objective

- Load a dataset into Python.
- Identify and handle missing values.
- Remove duplicate records.
- Convert incorrect data types.
- Clean inconsistent data.
- Export the cleaned dataset.

## Tasks Performed
